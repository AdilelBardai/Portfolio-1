Mijn kunst website.

Omdat ik van kunst en tekenen houd, vond ik het leuk om van mijn eerste ervaring om een website een heel eigen creatie te maken, met alleen maar eigen werk. Deze site bevat een set van mijn tekeningen en tools, waar je op een leuke manier doorheen kan bladeren.

Waar ik trots op ben in mijn werk:

- De website code

Dit was mijn eerste keer om te coderen en een website te maken. Het was veel werk om te doen, maar ook leuk om te zien dat mijn code goed werkt. Ik heb de foto's, linken en het idee van de inhoud, dus kan ik op de code concentreren. Ik heb veel tijd besteden om mijn code te bouwen en leren hoe alles met elkaar toepassen moet, daarom ben ik trots op mijn werk en zich zelf op het eind van dit vak.

- Gallery pagina

Ik ben blij met mijn website en mijn Gallery pagina's, want hun bestaan mooi uit en zoals ik hen wild. Duidelijk, mooi kleuren en het bestaat mooi uit dat was hoe ik om mijn tekeningen wil zien en ik heb dat gedaan met de html en css kennis die ik heb geleerd.

- Tools pagina

Ik ben tevreden over mijn tools pagina, want hij ziet er prachtig uit, de teksten staan goed met duidelijk uitleg en de knoppen naar de sites van de bronnen werkt heel goed.
