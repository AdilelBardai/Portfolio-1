ISGA prototype.

Dit project was mijn eerste keer om individueel te werken en iets creeren. Ik was bang van dat maar na een tijdje ging alles goed en ik had mijn werk online om te testen met mijn klas genoten. Het heeft me veel tijd gekost om de code goed te schrijven, want ik was in tegelijkertijd het code leren in andere vak.

Waar ik trots op ben in mijn werk:

- Wat is dit project?

Het doel van dit project was om een web app in correct html en css prototype te maken, waarin de belangrijskte informatie van de ISGA (International Student Guide Amsterdam) inhouden, dan kunnen de studenten de gids sneller, makkelijker en altijd te gebruiken.

- onderzoeken

Om door te gaan aan het werk moeten we eerst een onderzoek maken over het boek, zijn inhoud, zijn stijl, zijn layout en internationale studenten interviewen, daarna de resultaten verzamelen.

- Verbeteren

Ik heb trots op mijn alle pagina's en hoe werken ze zoals ik wild, zonder problemen met duidelijk afbeeldingen en het ontwerp die ik wild. Ik kon mijn alle pagina's volgens mijn idee ontwerpen en volgens de feedbacks verbeteren, dan had ik mijn eind versie, die heb ik trots op. Het was leuk om te zien, hoe mijn werk verbetert door het project.

- de code

Ik heb trots op mijn code want het was het eerste keer om veel schermen met één css file maken met verschillende stylen.Het was veel werk, maar ik heb ook goed resultaat, als een prototype gehad.
Note: Maat iPhone 6/7/8.

- Het proces

Het was een lange project met veel werk, nieuwe kennis en ervaringen, maar in het eind had ik mooie resutaat die kan prima werken.
